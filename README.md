# Protein Microstate Generator

Интерактивный веб-инструмент для визуализации и анализа конформационных состояний белковых цепей. Проект демонстрирует связь между аминокислотной последовательностью, вторичной структурой и термодинамическими параметрами.

## Возможности

- Визуализация трёхмерной структуры белка с цветовым кодированием вторичной структуры
- Интерактивное назначение состояний (α-спираль, β-лист, петля, β-поворот) отдельным остаткам
- Расчёт биофизических параметров в реальном времени
  - Термодинамические потенциалы (ΔG, ΔH, TΔS, Keq)
  - Торсионные углы φ/ψ
  - Радиус гирации (Rg) и расстояние между концами цепи
  - Площадь доступной растворителю поверхности (SASA)
  - Количество водородных связей и гидрофобных контактов
- Отображение карты Рамачандрана с распределением торсионных углов
- Визуализация ландшафта свободной энергии
- Симуляция методом Монте-Карло с регулируемым температурным параметром
- Экспорт скриптов для PyMOL

## Использование

1. Загрузить последовательность
   - Выбрать один из пресетов (убиквитин, полиаланин, β-пептид)
   - Или ввести собственную последовательность однобуквенным кодом
2. Нажать "Загрузить"
3. Для редактирования состояния отдельного остатка кликнуть на нём в панели "Цепочка"
4. Выбрать состояние из списка или использовать горячие клавиши 1-4
5. Наблюдать изменение структуры и биофизических параметров

## Управление

| Клавиша | Действие |
|---------|----------|
| Пробел  | Запуск/остановка анимации |
| →       | Шаг вперёд |
| R       | Случайные состояния |
| M       | Шаг Монте-Карло |
| + / -   | Приближение / отдаление |
| 1       | Назначить α-спираль |
| 2       | Назначить β-лист |
| 3       | Назначить петлю |
| 4       | Назначить β-поворот |

## Модель

Расчёты основаны на комбинации

- Эмпирических пропенсивностей аминокислот (адаптированные данные Chou-Fasman)
- Потенциалов вторичных структур
- Температурной зависимости через фактор Больцмана
- Упрощённой модели гидрофобных взаимодействий

Модель предназначена для образовательных целей и демонстрации принципов, а не для точных предсказаний структуры.

## Экспорт в PyMOL

Кнопка "Сгенерировать .pml скрипт" создаёт файл для PyMOL, который

- Раскрашивает остатки по типу вторичной структуры
- Добавляет анимацию вращения
- Включает рассчитанные параметры как комментарии
